<h3>课程介绍：</h3>
<p>汇编语法与汇编器的关系，类似于C/C++标准与C/C++编译器的关系，主要有两种语法，Intel 和AT&amp;T 汇编语法，最大的区别是后者是与平台无关的，前者只局限于Inter平台。特定CPU架构下汇编指令集和寄存器名称相当于汇编语法中的关键字，支持多平台的GAS汇编器会根据当前系统的CPU架构自动识别并转换成对应平台下的机器码。</p>
